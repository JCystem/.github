### ¡Encantado de verte aquí!  
JCystem Project, es un proyecto de código abierto desarrollador por Javier Caceres, que abarca desde Bots de WhatsApp y Telegram, hasta API´s de código abierto para cualquier uso. En su gran mayoría JCystem repara, modifica y distribuye códigos propios y de terceros.

JCystem Project también trabaja actualmente en el desarrollo de una Licencia de Software de código abierto propia para el manejo y uso legítimo de códigos publicados por JCystem Project y cualquier otro individuo que desee usarla.
